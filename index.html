About:

Cheatsheet

Action Command
Open nvimtree Leader (SPACE) + e
Toggle nvimtree sidebar ^ (CTRL) + n
Switch between open Tabs (Buffers) ] + b (OR) [b
Save Changes ^ (CTRL) + s (OR) :w (OR) wq!
